Les Paréo

Ce site est issu d’un projet d’apprentissage du code informatique né de la collaboration entre quelques étudiants du DU PaRéO et le club de code du CRI (Centres de Recherches Interdisciplinaires). Il s’agit d’une formation par des étudiants pour des étudiants pour se familiariser avec les bases du code, de l’intégration web et du web design.

Le but est de montrer que l’on n’a pas besoin d’un diplôme universitaire pour s’orienter. Ce site se présente sous la forme d’articles expliquant les activités du DU, comment se réorienter, pourquoi se réorienter, ...
